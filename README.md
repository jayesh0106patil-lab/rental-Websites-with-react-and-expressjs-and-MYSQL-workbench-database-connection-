# 🏠 Modern Web2/Web3 Rental Flat & Room Booking System

A modern full-stack rental booking platform built using:

- ⚛️ React.js
- 🎨 Bootstrap 5
- 🟢 Node.js + Express.js
- 🗄️ MySQL + MySQL Workbench

This project combines:
- Modern Web2 UI
- Web3-inspired futuristic styling
- Glassmorphism effects
- Responsive design
- Full-stack CRUD functionality
